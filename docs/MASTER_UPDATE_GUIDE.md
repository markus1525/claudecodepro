## Part 9: Common Issues and Solutions

### Issue: "I can't find Section X in my document"

**Solution:**
- Use Find (Ctrl+F) to search for section headings
- Check Table of Contents for structure
- Section may be named differently - search for key words

### Issue: "Tables from Markdown aren't formatting well in Word"

**Solution:**
1. Paste as plain text first
2. Select pasted content
3. Insert → Table → Convert Text to Table
4. Delimiter: Pipe (|) character
5. Apply Word table style

### Issue: "Code blocks look messy"

**Solution:**
1. Select code block
2. Font: Courier New or Consolas
3. Font size: 9-10pt
4. Indentation: 0.5"
5. Background: Light gray (optional)

### Issue: "Screenshots are too large/small"

**Solution:**
- Right-click image → Size and Position
- Width: 6.5" (for standard margins)
- Maintain aspect ratio: Checked
- Lock aspect ratio: Checked

### Issue: "Page numbers are wrong after updates"

**Solution:**
- Ensure all section breaks are "Next Page" not "Continuous"
- Update Table of Contents (right-click → Update Field)
- Check for manual page number overrides
